# Escape Goat 1 & 2 TAS Files

## Input File
Format for the input file is (Frames),(Actions)
ie) 35,R,J (For 35 frames, hold Right and Jump)

## Actions Available
- R = Right
- L = Left
- U = Up
- D = Down
- J = Jump / Confirm
- Z = Dash
- E = Henry
- H = Hat/Hammer
- S = Start
- C = Cancel
- X = Reset Room
